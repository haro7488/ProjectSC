# 라이브 이벤트 시스템

## 개요

라이브 서비스 운영을 위한 기간제 이벤트 시스템. 여러 이벤트가 동시에 운영되며, 각 이벤트는 독립적인 컨텐츠 컨테이너로 동작.

---

## 핵심 개념

| 개념 | 설명 |
|------|------|
| **LiveEvent** | 기간제 컨텐츠 컨테이너 (미션, 상점, 미니게임 등 포함 가능) |
| **EventDashboard** | 활성 이벤트 목록 UI (로비에서 진입) |
| **ILiveEventService** | 데이터 계층 추상화 (Local/Server 교체 가능) |

---

## 이벤트 특성

- **동시 운영**: 여러 이벤트가 동시에 활성화 가능
- **독립 컨테이너**: 각 이벤트는 자체 미션, 상점, 미니게임 포함 가능
- **기간 제한**: 시작/종료 시간, 반복 주기 설정
- **모듈 재사용**: 조건, 보상, 스케줄 등 공통 모듈 활용

---

## 이벤트 구성 요소

```
LiveEvent (컨테이너)
├── Missions        # 이벤트 전용 미션
├── Shop            # 이벤트 전용 상점
├── MiniGame        # 이벤트 전용 미니게임 (선택)
├── Rewards         # 보상 시스템
├── Schedule        # 기간/반복 설정
└── Navigation      # 바로가기 연동 (→ NavigationSystem)
```

---

## 데이터 계층

```
┌─────────────────────────┐
│   LiveEventManager      │
└───────────┬─────────────┘
            │
┌───────────▼─────────────┐
│  ILiveEventService      │  ← 인터페이스
└───────────┬─────────────┘
      ┌─────┴─────┐
      ▼           ▼
┌──────────┐ ┌──────────┐
│  Local   │ │  Server  │
│ Service  │ │ Service  │
│ (현재)    │ │ (추후)    │
└──────────┘ └──────────┘
```

---

## 연관 시스템

- **NavigationSystem**: 이벤트 내/외부 컨텐츠 이동 (설계 예정)
- **Quest**: 기존 퀘스트와 이벤트 미션 공통 모듈 공유 가능
- **Shop**: 기존 상점과 이벤트 상점 공통 모듈 공유 가능
