# ProjectSC 포트폴리오

## 개요

이 프로젝트는 **AI를 최대한 활용한 게임 개발 프로세스**를 보여주는 포트폴리오입니다.
서브컬쳐 수집형 RPG의 기반 구조를 설계하고 구현하면서, AI와의 협업 과정을 상세히 기록합니다.

## 포트폴리오 목적

1. **AI 협업 게임 개발** - Claude와의 대화를 통한 설계/구현 과정 시연
2. **의사결정 과정 공개** - 왜 이런 구조를 선택했는지 근거와 고민 기록
3. **실무 적용 가능성** - 실제 프로젝트에서 활용 가능한 패턴과 구조 제시
