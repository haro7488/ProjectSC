# 탐색 프로토콜

## 실행 조건
- 3개 이상 파일 탐색 필요
- "어디있지?", "찾아줘" 류 요청
- 영향 범위 파악 필요

## 위임 전략

| 탐색 유형 | 에이전트 | 모델 |
|-----------|----------|------|
| 파일/위치 탐색 | `Explore` | sonnet |
| 아키텍처 분석 | `code-explorer` | sonnet |
| 의존성 추적 | `code-explorer` | sonnet |

## 프롬프트 템플릿

```
[대상]에 대해 탐색해줘.

요구사항:
- 핵심 파일 3개 이내로 식별
- 주요 발견사항 bullet 3개
- 권장 다음 단계 1줄

완료 후 아래 형식으로 보고:
## 완료 보고
- 수행: (1줄)
- 결과: (파일 경로 또는 bullet 3개)
- 확인 필요: (있으면 기술)
```

## Serena 활용

단일 심볼 조회는 메인에서 직접:
- `find_symbol` - 클래스/메서드 위치
- `find_referencing_symbols` - 참조 추적
- `get_symbols_overview` - 파일 구조 파악
